## 环境搭建

* ### 全局安装angular/cli

```
npm install -g @angular/cli
```

* ### 查看版本

```
ng v
```

## 创建项目

```
ng new 项目名称
```

## 运行项目

```
ng serve 或 ng serve --open（自动打开浏览器）
```
